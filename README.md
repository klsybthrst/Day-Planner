# Day-Planner

#Usage
Plan you day with Day Planner! Keep track of your day with ease, using local storage to store re-occuring events like Lunch! Never lose track of time as the colour coded hours change throughout your day!

#Tools

Bootstrap
Google Fonts
Visual Studio Code
